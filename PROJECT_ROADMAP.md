# SinFleet ERP Roadmap

## Phase 0 - Foundation

- Initialize monorepo project structure.
- Add frontend and backend TypeScript setups.
- Add Prisma PostgreSQL schema with multi-tenant entities.
- Add SaaS documentation and AI memory files.

## Phase 1 - Authentication and Tenancy

- Super Admin login.
- Company Admin login.
- Driver/User login.
- JWT access and refresh token rotation.
- Tenant resolver middleware.
- Role and permission guards.

## Phase 2 - Company Panel MVP

- Dashboard metrics.
- Vehicle management.
- Driver management.
- Trip management.
- Expense and diesel entry.
- Client and payment tracking.

## Phase 3 - Super Admin Panel

- Company onboarding.
- Subscription management.
- Billing overview.
- Platform analytics.

## Phase 4 - Driver Mobile UI

- Simple driver dashboard.
- Trip status updates.
- Diesel upload.
- Expense upload with image support.

## Phase 5 - Operations and Intelligence

- Maintenance reminders.
- Document expiry alerts.
- Profit/loss analytics.
- Pending payment reports.
- WhatsApp notifications.

## Phase 6 - Future Platform

- Redis caching.
- WebSocket live updates.
- BullMQ background jobs.
- GPS tracking integrations.
- OCR for bills and receipts.
- AI analytics and forecasting.
- Mobile apps.
